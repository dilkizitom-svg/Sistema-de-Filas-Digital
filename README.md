## 📝 Sobre o Projeto
Este projeto foi desenvolvido como parte das **Jornadas Científicas do ISUTC** (4º ano de Engenharia Informática e de Telecomunicações). O objetivo principal é mitigar as longas filas de espera nas agências bancárias de Maputo, especificamente utilizando o **BCI** como banco piloto.

A solução permite que os clientes retirem senhas digitais via aplicativo móvel antes de chegarem à agência, otimizando a gestão do tempo e melhorando o fluxo de pessoas na cidade.

## ✨ Funcionalidades
- **Emissão de Senha Remota:** O usuário escolhe o serviço (Caixa/Depósito ou Atendimento) pelo telemóvel.
- **Notificações em Tempo Real:** Alertas inteligentes quando a senha está próxima de ser chamada.
- **Estimativa de Espera:** Cálculo baseado no fluxo atual da agência.
- **Dashboard da Agência:** Interface para os gestores monitorarem o volume de clientes.

## 🛠 Tecnologias Utilizadas
O projeto utiliza uma arquitetura moderna para garantir escalabilidade:

- **Mobile:** [Flutter](https://flutter.dev/) & [Dart](https://dart.dev/)
- **Backend:** [Spring Boot](https://spring.io/projects/spring-boot) (Java)
- **Base de Dados:** PostgreSQL
- **Controlo de Versão:** Git & GitHub
